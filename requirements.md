# Requirements — Eminent Website Renewal

## 1. Project Overview

The current Eminent website consists mostly of static content, with only a form-based inquiry feature. The goal of this renewal is to modernize the design, improve user experience, and enhance maintainability.

---

## 2. Objectives

- Refresh UI/UX with a modern design
- Ensure full responsive support (mobile/tablet)
- Improve performance and loading speed
- Optimize for SEO
- Increase inquiry conversion rate
- Enable easier content updates (CMS if applicable)

---

## 3. Required Materials

### 3.1 Content
- Company overview (About Us)
- Service/Product descriptions
- Existing website content (with notes on updates if needed)
- FAQ (if available)

### 3.2 Design Assets
- Logo files (PNG)
- Brand guidelines (colors, fonts, etc.)
- Reference websites (for design direction)
- List of issues with current site (if any)

### 3.3 Media
- Images (high resolution)
- Videos (if applicable)
- Icons or graphics

### 3.4 Functional Requirements
- Inquiry form (required fields and workflow)
- Admin panel (if needed)
- Blog/News section (optional)

### 3.5 Technical Information
- Domain and hosting details *(Eminent will provide)*
- Server access (if applicable) *(Eminent will provide)*
- Current tech stack (if any)

---

## 4. Key Features
